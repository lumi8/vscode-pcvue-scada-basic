# vscode-pcvue-scada-basic
Extension for PcVue SCADA Basic programming language in vscode.
This package provides syntax highlighting and snippets. 

## Features

Automatic recognition of .scb, .svb and .prg files.
Keywords
Strings
Function
Comments
Instructions
Snippets
